# Memory Card Game

A simple **Memory Card Game** built with **HTML and CSS**. The game challenges players to match pairs of cards by flipping them over, improving memory and concentration skills.

---

## Features

- Interactive card-flipping gameplay  
- Match pairs of cards to win  
- Simple and clean user interface  
- Fully responsive design  

---

## How to Play

1. Open `index.html` in your browser.  
2. Click on a card to flip it.  
3. Find its matching pair.  
4. Continue until all pairs are matched.  

---

## Technologies Used

- **HTML5** – Structure of the game  
- **CSS3** – Styling and animations  

---

## Future Improvements

- Add **JavaScript** to track moves and timer  
- Add **scoreboard** or high score system
